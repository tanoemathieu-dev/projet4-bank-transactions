# Projet 4 — Bank Transactions Analysis

## 📌 Objectif
Analyse multi-angles d'un portefeuille de transactions bancaires couvrant
la détection de fraude, le comportement des clients et la performance des transactions.

## 🗂️ Source des données
- Fichier CSV : `bank_transactions_data.csv`
- 2 512 transactions | 16 colonnes
- Données simulées de transactions bancaires américaines

## 🛠️ Outils utilisés
- **MySQL** — Nettoyage et préparation des données
- **Power BI** — Visualisation et dashboard interactif

## 📁 Contenu du repository
| Fichier | Description |
|---|---|
| `projet4_cleaning.sql` | Script complet de nettoyage MySQL |
| `bank_transactions_data.csv` | Dataset source |

## 🔍 Axes d'analyse
- **Fraude** : détection des comportements suspects via LoginAttempts et canaux
- **Comportement clients** : analyse par âge, occupation et canal préféré
- **Performance** : montants, durée des transactions et soldes
